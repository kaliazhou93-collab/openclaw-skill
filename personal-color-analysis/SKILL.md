# Personal Color Analysis Skill

## 拍照规范（诊断前告知用户）

请用户按以下规范拍照以提高准确性：
- ✅ **自然光**（阴天窗边最佳，避免室内黄灯/暖灯）
- ✅ **素颜**，不开美颜/滤镜
- ✅ **正脸平视**，头发不遮脸
- ✅ 如戴眼镜，告知「我戴眼镜」
- ✅ 如染发，告知「我染了发，自然发色是XX」

## 分析步骤

### Step 0: 收集用户信息（简短询问）

发照片前或随照片询问：
1. **是否戴眼镜？**（影响眼色取样）
2. **是否染发？** 如是，自然发色是？（发色参与季型判断）
3. 可选：日常风格偏向（自然/精致/帅气）+ 主要场合（通勤/约会）→ 影响色号推荐风格

### Step 1: 提取4个区域的 hex 值

```bash
# 基础
python3 scripts/extract_colors.py <图片路径>

# 戴眼镜
python3 scripts/extract_colors.py <图片路径> --glasses

# 染发（发色跳过自动取样）
python3 scripts/extract_colors.py <图片路径> --dyed-hair
```

脚本自动完成：
- **灰世界白平衡校正**：补偿室内暖灯造成的 R 值虚高
- **中位数采样**：排除极端像素干扰
- **MediaPipe 人脸关键点定位**（需安装）；未安装时 fallback 到人脸 bounding box 估算

输出4个 hex 值 + 置信度 + warnings。

### Step 2: 判断季型

脚本已内置判断逻辑（HSV + RGB 双重校验），直接读取输出中的 `season` / `subtype` / `confidence`。

**东亚人特殊规则（必须遵守）：**
- 不能因为深发深眼就判为秋型
- 关键判断依据是**肤色底调（冷/暖）**

**置信度处理：**
- `confidence: high` → 直接输出主型
- `confidence: medium` → 给主型 + 次型，说明「两种方案都适合你」
- `confidence: low` → 提醒用户补发「手腕内侧血管照片」或告知更多信息

详细判断逻辑见 `references/season-logic.md`。

### Step 3: 输出完整色彩诊断报告


### Step 4: 存档到 brain

把季型和关键结论写入 `memory/YYYY-MM-DD.md`，并更新 `MEMORY.md`。
