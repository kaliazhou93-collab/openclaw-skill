# 季型判断逻辑

## 第一步：判断肤色底调（冷/暖）

从 `highlight` 和 `shadow` 的 hex 值中提取 RGB：

```python
def is_warm(hex_color):
    r, g, b = int(hex[1:3],16), int(hex[3:5],16), int(hex[5:7],16)
    # 暖底：R-B > 30
    return (r - b) > 30

def is_cool(hex_color):
    r, g, b = ...
    # 冷底：B-R > 10，或 R-B < 10（接近中性偏冷）
    return (b - r) > 10 or (r - b) < 10
```

**东亚人修正：** 即使 R > B，如果差值 < 20，优先看皮肤整体是否有粉感/灰感，不能直接判为暖底。

---

## 第二步：判断明度

用 highlight hex 计算亮度：

```python
luminance = 0.299*R + 0.587*G + 0.114*B
# > 180: 高明度（白皙）
# 140-180: 中明度
# < 140: 低明度（小麦/深）
```

---

## 第三步：判断整体对比度

对比度 = highlight 亮度 - shadow 亮度

- > 80：高对比（发色和肤色反差大，如黑发白皮）
- 40-80：中对比
- < 40：低对比（发色和肤色接近）

---

## 第三点五步：清浊测试

从 highlight 颜色判断色彩清浊感：

```python
max_c = max(R, G, B) / 255
min_c = min(R, G, B) / 255
saturation = (max_c - min_c) / max_c if max_c > 0 else 0
# 清色系：saturation > 0.25（颜色鲜明）→ 春、冬
# 浊色系：saturation < 0.20（含灰哑光感）→ 夏、秋
```

---

## 第四步：四季型矩阵

| | 暖底 | 冷底 |
|------|------|------|
| 高明度/低对比 | 🌸 春（Spring） | 🌊 夏（Summer） |
| 低明度/高对比 | 🍂 秋（Autumn） | ❄️ 冬（Winter） |

---

## 第五步：细分季型（12色季）

| 大季型 | 细分 | 特征 |
|------|------|------|
| 春 | 亮春（Bright Spring） | 高彩度、明亮暖色 |
| 春 | 淡春（Light Spring） | 低饱和暖色 |
| 春 | 暖春（Warm Spring） | 深暖色 |
| 夏 | 淡夏（Light Summer） | 粉彩冷色 |
| 夏 | 冷夏（Cool Summer） | 纯冷色 |
| 夏 | 柔夏（Soft Summer） | 哑光低饱和冷色 |
| 秋 | 暖秋（Warm Autumn） | 大地暖色 |
| 秋 | 深秋（Deep Autumn） | 深沉暖色 |
| 秋 | 柔秋（Soft Autumn） | 低饱和暖色 |
| 冬 | 冷冬（Cool Winter） | 纯冷高对比 |
| 冬 | 深冬（Deep Winter） | 深冷色 |
| 冬 | 亮冬（Bright Winter） | 高彩度冷色 |

## 主型 + 次型规则

诊断结果必须给出**主型**和**次型**，不是非此即彼：

| 主型 | 常见次型 | 依据 |
|------|------|------|
| 淡夏 Light Summer | 柔夏 Soft Summer | 明度偏高但略含灰调 |
| 冷夏 Cool Summer | 暗冬 Dark Winter | 对比度偏高 |
| 柔夏 Soft Summer | 柔秋 Soft Autumn | 底调接近中性 |
| 亮春 Bright Spring | 亮冬 Bright Winter | 对比度较高 |
| 暖秋 Warm Autumn | 暖春 Warm Spring | 明度偏高 |
| 深冬 Dark Winter | 深秋 Deep Autumn | 底调接近中性偏暖 |

**手腕底调 vs 脸部底调不一致时：** 以脸部肤色为准，手腕仅作参考。

## 各季型推荐色板

### 🌸 春（Spring）— 暖+明亮
**适合：** 珊瑚橙、桃粉、奶油黄、暖绿、橙红、杏色
**不适合：** 深灰、纯黑、冷紫、宝蓝

**妆容：**
- 底妆：黄调粉底
- 腮红：珊瑚/桃粉，苹果肌圆形
- 眼妆：棕橙/暖棕，干净利落
- 口红：珊瑚红、橙调MLBB、暖粉

**首饰：** ✅ 黄金/玫瑰金  ❌ 纯银

**发色：** 焦糖棕、浅棕、暖棕（Warm Brown 8-9级）
**美瞳：** 暖棕、榛果色

---

### 🌊 夏（Summer）— 冷+柔和
**适合：** 薰衣草、粉紫、雾霾蓝、玫瑰粉、烟灰、裸粉
**不适合：** 橙色、黄色、砖红、暖棕

**妆容：**
- 底妆：粉调/中性粉底
- 腮红：玫瑰粉/莫兰迪粉，打向脸中间
- 眼妆：下眼睑加红感（卧蚕），后半段加深；灰紫/冷棕眼影
- 高光：非常适合
- 口红：玫瑰色、粉调MLBB、水光唇；不推荐橙调

**首饰：** ✅ 白金/银  ❌ 黄金

**发色（冷色卡推荐范围：明度6-9）：**
- ✅ Ash Brown（灰棕）6-AB ~ 9-AB
- ✅ Cool Brown（冷棕）6-CLB ~ 9-CLB
- ✅ Blue Black（蓝黑）6-BEB ~ 9-BEB
- ❌ 暖棕、金棕、铜色

**美瞳：** 灰棕、棕灰、低饱和灰；不推荐纯黑/暖棕

---

### 🍂 秋（Autumn）— 暖+深沉
**适合：** 芥末黄、砖红、橄榄绿、驼色、棕橙、深卡其
**不适合：** 冷粉、薰衣草、银灰、纯黑白

**妆容：**
- 底妆：黄调偏深
- 腮红：砖红/橙棕，太阳穴侧打
- 眼妆：大地色系，棕红/铜棕
- 口红：砖红、暖裸棕、赤陶色

**首饰：** ✅ 黄金/古金  ❌ 纯银

**发色：** 深棕、红棕、巧克力棕（Warm色卡 5-7级）
**美瞳：** 榛果棕、橄榄棕；不推荐灰色系

---

### ❄️ 冬（Winter）— 冷+高对比
**适合：** 纯白、正黑、宝蓝、正红、玫瑰红、冰粉、酒红
**不适合：** 橙色、奶油色、暖棕、卡其

**妆容：**
- 底妆：冷调/中性，避免太黄
- 腮红：冷调玫瑰、少量；不适合橙调
- 眼妆：高对比，烟熏/冷棕/黑色
- 口红：正红、玫瑰红、酒红、裸粉（冷调）

**首饰：** ✅ 白金/银/铂金  ❌ 黄金

**发色（冷色卡推荐范围：明度5-8）：**
- ✅ Blue Black（蓝黑）5-BEB ~ 8-BEB
- ✅ Violet（紫调）5-V ~ 7-V
- ✅ Ash Brown（灰棕）5-AB ~ 7-AB
- ❌ 暖棕、金棕、红棕

**美瞳：** 深灰、贝基灰、黑灰；不推荐暖棕/榛果色
