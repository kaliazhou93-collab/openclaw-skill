# Personality Receipt · 人格运行小票

> 你不需要填问卷。你的 AI 早就看透你了。

---

## 是什么

一个让 AI 基于**长期相处记忆**生成「人格运行小票」的 Agent Skill。

不是 5 道题测出你是什么人。而是——

**你用了 AI 三个月、半年、一年之后，它回过头来告诉你：我眼里的你，是这样的。**

---

## 设计理念

```
传统人格测试：  回答 50 道题 → 给你一个标签
MBTI 热帖：   千篇一律的问题 → 千篇一律的结果

人格小票：    你什么都不用做。
             你的 AI 已经在数百次对话中观察了你——
             你怎么做决定、你在压力下怎么反应、
             你和别人协作时的第一反应是什么。

             它比大部分人都更了解你。
             现在，它把观察结果打印出来了。
```

---

## 为什么它更真实

| 传统测试 | 人格小票 |
|---------|---------|
| 你告诉测试你是谁 | AI 从行为中推断你是谁 |
| 你可以"演"一个理想的自己 | AI 看到的是你真实的反应模式 |
| 千人一面的标准化题目 | 基于你独有的对话历史 |
| 测一次就过时了 | 每次生成都反映最新的你 |

## 使用方式

对你的 AI 说：

> 「帮我生成人格小票」

如果你们相处够久，它会直接出票。
如果还不够了解你，它会先聊几句——不是问卷，是真正的对话。

---

## 安装

将 `SKILL.md` 放入你的 AI Agent 工作目录，或按照你使用的 Agent 框架方式加载。

兼容：OpenClaw / Claude / 其他支持 SKILL.md 格式的 Agent。
